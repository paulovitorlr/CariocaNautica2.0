# Carioca Náutica 2.0

Landing Page institucional desenvolvida para a **Carioca Náutica**, empresa especializada na instalação de pisos náuticos para embarcações.

O projeto tem como objetivo fortalecer a presença digital da empresa, apresentar seus serviços de forma profissional e aumentar a conversão de visitantes em clientes por meio de uma experiência moderna, rápida e responsiva.

---

## 🎯 Objetivos

- Apresentar os serviços da Carioca Náutica;
- Destacar os diferenciais dos pisos náuticos;
- Gerar leads qualificados através de formulários e CTAs estratégicos;
- Reforçar a credibilidade da marca;
- Oferecer uma experiência otimizada para dispositivos móveis e desktop.

## 🏗️ Arquitetura

O projeto foi estruturado utilizando uma arquitetura baseada em componentes, priorizando:

- Escalabilidade;
- Reutilização de código;
- Organização por domínio;
- Facilidade de manutenção;
- Separação de responsabilidades.

```
src/
├── core/
│   ├── data/
│   └── interfaces/
├── pages/
│   └── home/
│       └── sections/
├── shared/
└── styles/
```
